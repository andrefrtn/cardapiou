# 🍽️ Cardapiou

Sistema de gerenciamento de cardápio desenvolvido para aprender CRUD com Java e Spring Boot.

## 👨‍💻 Desenvolvedor

**André Fortini**

## 📋 Sobre o Projeto

Aplicação fullstack de gerenciamento de itens de cardápio, permitindo cadastrar, visualizar, editar e deletar pratos com nome, preço e imagem.

## 🚀 Tecnologias

**Frontend**
- React + TypeScript
- Vite
- TanStack Query (React Query)
- Axios

**Backend**
- Java + Spring Boot
- Spring Data JPA
- Lombok
- Banco de dados relacional

## ⚙️ Funcionalidades

- ✅ Listar itens do cardápio
- ✅ Cadastrar novo item
- ✅ Editar item existente
- ✅ Deletar item

## ⚙️ Pré-requisitos

- Node.js instalado
- Java 17+ instalado
- Banco de dados configurado em `Back-end/src/main/resources/application.properties`

## 🖥️ Como rodar

**Backend**
```bash
cd Back-end
.\mvnw.cmd spring-boot:run
```

**Frontend**
```bash
cd Front-end
npm install
npm run dev
```

Acesse: `http://localhost:5173`
